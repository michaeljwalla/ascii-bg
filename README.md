# ascii-bg
utilizes `ascii-image-converter` to parse and set a window/instance with an ascii form of an image. caches as well

wrote in a day bc i was bored and wanted to have it & use uv & larp

## dependencies
- **[`ascii-image-converter`](https://github.com/TheZoraiz/ascii-image-converter)** - the hard work
- **[`uv`](https://github.com/astral-sh/uv)** - more hard work

## installation
```bash
git clone https://github.com/michaeljwalla/ascii-bg.git
cd ascii-bg
chmod +x ./install.sh && ./install.sh
```
